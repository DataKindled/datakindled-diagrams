Data types & table formats
Unstructured
Documents, images, audio, and video stored in Snowflake and processable using Cortex AI functions and Document AI.
Semi-structured
JSON, Parquet, Avro, ORC, and XML stored natively in Snowflake's VARIANT type — queryable without flattening the structure.
Structured
Traditional tabular data — stored in Snowflake's compressed columnar format with AES-256 encryption and micro-partition optimization.
Iceberg tables
Open table format stored in customer-managed cloud storage — Snowflake provides compute and governance while the data remains in open format.
Snowflake tables
Fully Snowflake-managed tables — optimized columnar storage with automatic clustering, compression, and Time Travel built in.
Hybrid tables
Row-based tables optimized for low-latency transactional reads and writes — enabling OLTP workloads alongside analytical queries.
Snowflake Postgres
PostgreSQL-compatible interface for Snowflake — allowing Postgres-native applications and tools to connect directly to Snowflake.
